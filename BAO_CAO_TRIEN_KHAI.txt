VAN DE 1: Flask-SocketIO Greenlet Exhaustion (DA FIX)
--------------------------------------------------------------

Trieu chung:
  - Camera bai xe hien thi tot, nhung camera cong KHONG hien thi
  - Giao dien web BI TREO CUUNG sau khi mo dashboard
  - Khong the bam nut logout hay tuong tac gi duoc
  - Khi mo tab test endpoint /test_gate_feed, trinh duyet bi load vo han

Nguyen nhan goc (Root Cause):
  Flask-SocketIO su dung eventlet - mot thu vien dua tren greenlet (khong phai thread).
  Trong che do mac dinh, eventlet chi cap mot so luong greenlet han che.

  Khi trinh duyet mo dashboard, no gui 2 request MJPEG stream cung luc:
    1. /video_feed (camera bai xe)
    2. /video_feed_gate (camera cong)

  Moi MJPEG stream giu mot ket noi mo vinh vien (long-lived connection).
  Moi ket noi chiem 1 greenlet. Chi co 2 greenlet cho streaming, Flask/SocketIO
  KHONG CON greenlet nao de xu ly cac request khac (API, page navigation, logout).

Cach sua:
  File: main.py

  Thay doi:
    socketio = SocketIO(app, cors_allowed_origins="*", async_mode='threading',
                         engineio_logger=False, logger=False)

  Them async_mode='threading' de SocketIO dung thread pool thay vi greenlet.
  Moi request/connection chay trong thread rieng, khong bi gioi han boi so luong
  greenlet co dinh.

VAN DE 3: Flutter Mobile App Compilation Errors (DA FIX)
--------------------------------------------------------------

Trieu chung:
  Flutter build that bai voi 2 loi Dart:

  1. Error: Can't have modifier 'static' here.
     File: lib/services/notification_service.dart, line 13

  2. Error: The getter 'config' isn't defined for the type 'SocketService'.
     File: lib/services/socket_service.dart, lines 21 va 34

Cach sua:

  1. notification_service.dart (dong 13):
     Cu: static const _seenKeysPref = 'shown_notif_keys';
     Moi: const _seenKeysPref = 'shown_notif_keys';

     Ly do: Dart khong cho phep 'static' o cap top-level trong class.
            Chi cho phep trong class body.

  2. socket_service.dart (dong 3, 21, 34):
     Cu: import '../config.dart';
         config.apiBaseUrl, ...

     Moi: import '../config.dart' as config;
          config.apiBaseUrl, ...

     Ly do: Tranh xung dot ten giua import 'config.dart' voi ten class 'Config'
            (neu co). Dung prefix 'config.' ro rang hon.
